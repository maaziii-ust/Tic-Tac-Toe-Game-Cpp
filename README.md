# Tic Tac Toe Game (C++)

## Overview

Tic Tac Toe is a console-based two-player game developed in C++. Players take turns placing their markers on a 3×3 board, and the first player to complete a row, column, or diagonal wins the game.

## Features

* Two-Player Gameplay
* Marker Selection (X or O)
* Win Detection
* Tie Detection
* Input Validation
* Board Management
* Interactive Console Interface

## Technologies Used

* C++
* Arrays
* Functions
* Conditional Statements
* Loops

## Game Rules

1. Player 1 chooses X or O.
2. Players take turns placing markers.
3. Complete a row, column, or diagonal to win.
4. If all slots are filled without a winner, the game ends in a tie.
